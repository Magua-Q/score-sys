# 响应系统如何实现

### 副作用

通过一个例子来理解什么是副作用, 代码如下:

```
 let val = 1

 function effect () {
    var = 2
 }

```

当函数执行的时候会修改全局变量的值，这就是一个简单的副作用。

## 响应式数据的基本实现

1. 将副作用函数存储到"桶"中
2. 将副作用函数从"桶"中取出并执行

思考：
如果拦截一个对象属性的读取和设置操作

实现：

```js
// 副作用函数
function effect() {
  console.log(data.text);
}
// 原始数据
const data = {
  text: "test",
};
// 存储副作用函数的桶
const bucket = new Set();

// 对原始数据代理
const obj = new Proxy(data, {
  // 拦截读取操作
  get(target, key) {
    // 将副作用函数存储到桶中
    bucket.add(effect);
    return target[key];
  },

  // 拦截设置操作
  set(target, key, newVal) {
    target[key] = newVal;
    // 把副作用函数从"桶"中取出并执行
    bucket.forEach((fn) => fn());
    return true;
  },
});
```

总结一下，响应是的工作流程：
当读取操作发生时，将副作用函数收集到"桶"中;
当设置操作发生时，将副作用函数函数从"桶"
